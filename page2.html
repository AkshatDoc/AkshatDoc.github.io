DIY CNC

After using my school's CNC, I gained an obsession with automated macheniry. I decided that I wanted to make my own CNC. I went through many many different design iterations, using different technologies, like sliders or rods and I finally came up with one design. This design allows people to change the head from a router bit to a laser bit and much much more.

ChairBot

In my grade 12 Tech Design class, I was given the opportunity to design a new outreach robot for our team. Previously, we had a plastic chair on top of a small robot, which worked really well. The main problem I had with it was how it looked, so I decided to change that up. At first, I wanted to put an office chair on it but then I realized that it would be much safer is was a leaned back chair, like an Adirondack! I designed this along with the robot for 2 weeks and am currently in the build stage of the project! Photos coming soon

DIY Home Print Server
This is a screenshot of a print server I set up in my house. I used an old Optiplex750 as the PC and an old Bell Router. I set up a new wireless signal from the Router. This allowed our family to connect to the Wi-Fi and print! For this project, I needed Networking Skills and PC skills which I gained from the Cisco ITE Course
Next Steps: Add a Router that can act as a bridge to allow printing on main house Wi-Fi.
Dream Automated Room
This is a schematic/drawing of my ideal SMART room integrated with the latest home automation technology. This includes Smart Lights that dim and turn off on voice, a locking mechanism on my door using NFC or RFID, Camera syste that detects me and intruders, Bluetooth speaker system which can be controlled from any verified device and SMART ALARM which would allow me to use light and sound to wake me in the morning
Next Steps: Begin assessing what is realistic with my budget and available technology and begin prototyping SMART Security with an Arduino
